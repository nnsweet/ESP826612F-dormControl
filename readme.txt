使用说明：
将舵机、oled0.96寸屏幕、红外led连接好对应引脚后，将nodemcu通电。首先会nodemcu广播wifi，初始名称为“喜气洋洋”，
连接广播的wifi后，对其配置网络（此网络用于将nodemcu配网，用于手机端点灯科技APP使用）。

功能：
1.舵机开关灯
2.增添了屏幕播放ikun
3.增加空调开关与温度调节
注意:
1.需要将点灯科技的配置信息覆盖在自己的设备上
2.本次需要烧录程序和闪存。闪存内容为ikun跳舞，oled输出的存储信息，烧录闪存的时候需要安装版本低于2.0的arduino软件，



硬件材料：
1*ESP8266nodeMCU
1*三极管
1*红外led
2*舵机
1*oled0.96寸
